我们会把常用的工具集放入 npm 包中，方便集中维护, 所有工具集都会放入 klg 这个域中

## @klg/web
web 相关，主要是各种中间件

## @klg/utils

收集了 Kalengo 后端开发常用的工具类，目前有
### DateUtil
日期计算，节假日

### NumberUtil
主要处理 0.1 + 0.2 = 0.30000000000000004 问题

### Logger
logger 工具，可以显示logger所在代码位置

### StringUtil(TODO)

## @klg/mongoose
Mongodb 相关的辅助工具，包括连接配置，测试数据 Mock 等
